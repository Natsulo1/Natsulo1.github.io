Seção 2 — Ranking de Tecnologias

  1. HTML

    Linguagem de marcação responsável pela estruturação das páginas web

  2. CSS

    Utilizado para definir a aparência visual e o layout das páginas

  3. JavaScript

    Linguagem que adiciona interatividade e dinamismo às aplicações web

  4. React

    Biblioteca amplamente utilizada para construção de interfaces modernas e componentes reutilizáveis

  5. Node.js

    Ambiente de execução JavaScript que permite criar aplicações backend escaláveis

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web
CSS
Linguagem utilizada para estilizar documentos HTML
JavaScript
Linguagem de programação voltada para interatividade na web.
Pode ser executada tanto no navegador quanto no servidor
DOM
Representação em árvore dos elementos de uma página HTML
API
Conjunto de regras que permite a comunicação entre sistemas.
Facilita a integração entre aplicações diferentes
HTTP
HyperText Transfer Protocol
Protocolo utilizado para transferência de dados entre cliente e servidor na Web
URL
Uniform Resource Locator
Endereço utilizado para localizar recursos na Internet
DNS
Sistema responsável por traduzir nomes de domínio em endereços IP
